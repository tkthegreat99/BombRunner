# Boom Runner (붐 러너)

**Boom Runner**는 Unity 기반의 탑다운 3D 멀티플레이 캐주얼 생존 파티 게임 프로젝트입니다.

현재는 초기 개발 단계이며, 로컬 코어 루프와 Steam 기반 멀티플레이 구조를 구축 중입니다.

---

## Tech Stack

### Engine

- Unity
- Universal Render Pipeline (URP)
- New Input System
- Cinemachine
- AI Navigation / NavMesh

### Architecture

- VContainer
- UniTask
- R3
- DOTween

### Multiplayer

- Unity Netcode for GameObjects
- Host-authoritative multiplayer
- Steam P2P Transport 예정
- Facepunch.Steamworks 예정

### Steam Integration

- Steam Lobby
- Steam Invite
- Steam Overlay
- Steam Rich Presence
- Steam Achievements
- Steam Voice API
- Steam Workshop

### Platform

- PC
- Steam
